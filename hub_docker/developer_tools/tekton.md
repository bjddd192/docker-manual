# tekton

Tekton 是一个功能强大且灵活的Kubernetes 原生开源框架，用于创建持续集成和交付（CI/CD）系统。Tekton不能使你立即获得CICD的能力。但是基于Tekton可以设计出各种花式的构建部署流水线。得益于Tekton良好的抽象，这些设计出的流水线可以作为模板在多个组织，项目间共享。 

[官网](https://tekton.dev/)

[Tekton Catalog](https://github.com/tektoncd/catalog)
